The simplest Ajax function offered by jQuery is load(). This functions loads an HTML file into a specific element on the page.

To use the load() function, you first use a jQuery selector to identify the element on the page where the requested HTML should go; then call the load() function and pass the URL of the page you wish to retrieve.
