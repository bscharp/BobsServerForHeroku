JavaScript libraries like jQuery solve a lot of the problems with JavaScript programming - including pesky browser incompatibilities. Since events are so integral to JavaScript programing, it's better to think of event programming as a 3 step jQuery process:

1. Select one or more elements with a CSS selector.
$('#menu');

2. Assign an event.
$('#menu').click();

3. Pass a defined function to the event
$('#menu').click(startSlideShow);

or pass an anonymous function to the event
$('#menu').click(function() {
-- javascript code goes here --
}); // end click
